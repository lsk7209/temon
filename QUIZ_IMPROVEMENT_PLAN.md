## 🚨 문제점

### 발견된 문제 질문 패턴
- "{주제} 사용 감정은?"
- "{주제} 사용 이유는?"
- "{주제} 사용 후기는?"
- "{주제} 선택 방식은?"
- "{주제} 선택 기준은?"
- "{주제} 패턴은?"
- "{주제} 빈도는?"
- "{주제} 계획은?"

## 📋 개선 작업 계획

### 1단계: 문제 퀴즈 식별
- [x] 템플릿 질문 패턴 검색 완료
- [ ] 문제가 있는 모든 퀴즈 목록 작성
- [ ] 우선순위 결정

### 2단계: 질문 재작성
- [ ] 각 퀴즈의 질문을 재미있고 유쾌하게 재작성
- [ ] 바이럴 될 수 있는 상황으로 변경
- [ ] 공감대를 형성하는 일상적 상황으로 변경

### 3단계: 검증
- [ ] 재작성된 질문이 재미있는지 확인
- [ ] 공유하고 싶은 내용인지 확인
- [ ] 바이럴 될 수 있는지 확인

## 📝 개선 체크리스트

각 질문을 개선한 후 확인:

- [ ] **재미있는가?** - 읽고 웃음이 나오는가?
- [ ] **구체적인가?** - 상황이 명확하게 그려지는가?
- [ ] **공감되는가?** - "아, 나도 그런 적 있어!"라고 생각되는가?
- [ ] **공유하고 싶은가?** - 친구에게 보여주고 싶은가?
- [ ] **바이럴 될 수 있는가?** - SNS에 올리고 싶은가?
- [ ] **템플릿 패턴을 사용하지 않았는가?** - "{주제} 사용 감정은?" 같은 패턴이 없는가?
- [ ] **일상적이고 친근한가?** - 누구나 경험할 수 있는 상황인가?
- [ ] **유쾌하고 긍정적인가?** - 부정적이거나 무거운 내용이 아닌가?
