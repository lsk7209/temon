## 📝 참고사항

### 파일 구조
- **설정 파일**: `lib/tests-config.ts` - 모든 퀴즈 메타데이터 정의
- **레지스트리**: `tests-registry.json` - SEO/사이트맵용 퀴즈 목록
- **페이지 경로**: `app/tests/{quiz-id}/` - 각 퀴즈별 페이지 디렉토리

### 퀴즈 페이지 구조
```
app/tests/{quiz-id}/
├── page.tsx              # 퀴즈 소개 페이지
└── test/
    ├── page.tsx          # 퀴즈 진행 페이지
    └── result/
        └── page.tsx      # 결과 페이지
```

### 인기 퀴즈 (popular: true)
- `coffee-mbti`
- `ramen-mbti`

### 신규 퀴즈 (new: true)
- `kdrama-mbti`
- `snowwhite-mbti`
- `kpop-idol`
- 대부분의 생활 카테고리 퀴즈들

### 특이사항
- `lunch-style` 퀴즈는 삭제되었고 `lunch-decider`로 대체됨

## 🔧 개발 시 참고

1. **새 퀴즈 추가 시**:
   - `lib/tests-config.ts`의 `ALL_TESTS` 배열에 추가
   - `app/tests/{quiz-id}/` 디렉토리 구조 생성
   - `tests-registry.json`에 자동 등록됨 (스크립트 실행 시)

2. **카테고리 추가**:
   - `lib/tests-config.ts`의 `CATEGORIES` 배열에 자동 반영됨

3. **퀴즈 ID 규칙**:
   - kebab-case 사용
   - 의미 있는 이름 사용 (예: `coffee-mbti`, `love-reaction`)
