## 📝 개발 가이드

### 퀴즈 ID 규칙
- kebab-case 사용 (예: `food-temperature`, `morning-greeting`)
- 의미 있는 이름 사용
- 기존 퀴즈와 중복 확인 필수

### 질문 구성
- 표준 12문항
- 각 질문은 2개 선택지
- MBTI 16유형 기반 (E/I, S/N, T/F, J/P)

### 파일 구조
```
app/tests/{quiz-id}/
├── page.tsx              # 퀴즈 소개 페이지
└── test/
    ├── page.tsx          # 퀴즈 진행 페이지
    └── result/
        └── page.tsx      # 결과 페이지
```

### 결과 페이지 구성
- 16개 MBTI 유형별 상세 설명
- 각 유형별 특징, 추천, 성장 포인트, 궁합 포함
