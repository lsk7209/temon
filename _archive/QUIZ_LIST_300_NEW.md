- 각 퀴즈는 12개 질문으로 구성
- 16개 MBTI 결과 타입 제공
- 결과 페이지는 풍부한 콘텐츠 포함
- 기존 퀴즈와 중복되지 않도록 주의
- 각 퀴즈마다 고유한 색상 스키마 적용
