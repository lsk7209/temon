# 퀴즈 질문 작성 가이드라인 - 바이럴 & 재미있게 만들기

## 🎯 테몬 퀴즈의 핵심 철학

**"사람들이 공유하고 싶어하는 재미있는 퀴즈"**

- 재미있고 유쾌해야 함
- 바이럴 될 수 있어야 함
- 공유하고 싶게 만들어야 함
- 웃음과 공감을 유발해야 함

---

## 🚨 절대 금지 사항 (발견 시 즉시 수정)

### ❌ 템플릿 질문 패턴 (절대 사용 금지)

```
BAD: "{주제} 사용 감정은?"
BAD: "{주제} 사용 이유는?"
BAD: "{주제} 사용 후기는?"
BAD: "{주제} 선택 방식은?"
BAD: "{주제} 선택 기준은?"
BAD: "{주제} 변경은?"
BAD: "{주제} 패턴은?"
BAD: "{주제} 빈도는?"
BAD: "{주제} 계획은?"
BAD: "{주제} 준비는?"
BAD: "{주제} 경험 활용은?"

```

**이런 질문들은 즉시 삭제하고 재작성해야 합니다!**

## ✅ 바이럴 질문 작성 원칙

### 1. 구체적이고 재미있는 상황 묘사

#### ❌ 나쁜 예시
```
"화장실 사용 감정은?"
"화장실 사용 이유는?"
"화장실 사용 후기는?"
```

#### ✅ 좋은 예시
```
"화장실에서 핸드폰을 떨어뜨렸을 때"
"화장실 문이 잠겨있는데 급할 때"
"화장실에서 이상한 소리가 들릴 때"
"화장실 휴지가 다 떨어졌을 때"
"화장실에서 옆 칸 사람이 이상한 소리를 낼 때"
```

### 2. 공감대를 형성하는 일상적 상황

#### ❌ 나쁜 예시
```
"요리 시간 선택 방식은?"
"요리 시간 선택 기준은?"
"요리 시간 패턴은?"
```

#### ✅ 좋은 예시
```
"요리 중 타이머가 울렸는데 아직 안 익었을 때"
"요리 중 친구가 와서 맛보자고 할 때"
"요리 중 재료를 깜빡 잊고 안 넣었을 때"
"요리 중 불이 꺼졌을 때"
"요리 중 옆집에서 이상한 냄새가 날 때"
```

### 3. 웃음과 재미를 유발하는 상황

#### ❌ 나쁜 예시
```
"플레이팅 선택 방식은?"
"플레이팅 선택 기준은?"
"플레이팅 패턴은?"
```

#### ✅ 좋은 예시
```
"요리한 음식이 예쁘지 않게 나왔을 때"
"요리한 음식을 사진 찍으려는데 맛이 없을 때"
"요리한 음식을 친구에게 보여줬는데 반응이 없을 때"
"요리한 음식이 너무 예뻐서 먹기 아까울 때"
"요리한 음식을 인스타에 올리려는데 사진이 안 나올 때"
```

### 4. 공유하고 싶게 만드는 재미있는 상황

#### ❌ 나쁜 예시
```
"요리 후 정리 방식은?"
"요리 후 정리 선택 기준은?"
"요리 후 정리 패턴은?"
```

#### ✅ 좋은 예시
```
"요리 후 주방이 전쟁터가 되었을 때"
"요리 후 설거지가 산더미처럼 쌓였을 때"
"요리 후 정리하다가 또 요리하고 싶을 때"
"요리 후 정리하다가 깜빡 잊고 안 닦은 그릇을 발견했을 때"
"요리 후 정리하다가 배가 또 고파질 때"
```

## 🎨 바이럴 질문 작성 템플릿

### 템플릿 1: 예상치 못한 상황
```
"{일상 상황}에서 {예상치 못한 일}이 일어났을 때"
"{일상 상황} 중에 {재미있는 상황}이 발생했을 때"
"{일상 상황}에서 {당황스러운 상황}을 만났을 때"
```

**예시:**
- "마트에서 장을 보다가 예산을 초과했을 때"
- "식당에서 주문한 음식이 예상과 완전히 다를 때"
- "요리 중 불이 꺼졌는데 재료는 다 넣었을 때"

### 템플릿 2: 선택의 순간
```
"{상황}에서 {선택 A}와 {선택 B} 중 하나를 해야 할 때"
"{상황}에서 {선택 A}를 하려는데 {장애물}이 있을 때"
"{상황}에서 {선택 A}를 하면 {결과 A}, {선택 B}를 하면 {결과 B}일 때"
```

**예시:**
- "배가 고픈데 맛집 줄이 너무 길 때"
- "배가 고픈데 배달비가 너무 비쌀 때"
- "배가 고픈데 냉장고에 아무것도 없을 때"

### 템플릿 3: 감정과 반응
```
"{상황}에서 {감정}을 느낄 때"
"{상황}에서 {반응}을 보일 때"
"{상황}에서 {행동}을 하게 될 때"
```

**예시:**
- "맛있는 음식을 먹다가 실수로 떨어뜨렸을 때"
- "맛있는 음식을 먹다가 배가 터질 것 같을 때"
- "맛있는 음식을 먹다가 친구가 와서 나눠야 할 때"

### 템플릿 4: 일상의 재미있는 순간
```
"{일상 행동} 중에 {재미있는 일}이 일어났을 때"
"{일상 행동} 중에 {예상치 못한 상황}을 만났을 때"
"{일상 행동} 중에 {당황스러운 상황}이 발생했을 때"
```

**예시:**
- "요리 중에 재료를 깜빡 잊고 안 넣었을 때"
- "요리 중에 타이머가 울렸는데 아직 안 익었을 때"
- "요리 중에 불이 꺼졌는데 재료는 다 넣었을 때"

## 📝 질문 작성 체크리스트

각 질문을 작성한 후 반드시 확인:

- [ ] **재미있는가?** - 읽고 웃음이 나오는가?
- [ ] **구체적인가?** - 상황이 명확하게 그려지는가?
- [ ] **공감되는가?** - "아, 나도 그런 적 있어!"라고 생각되는가?
- [ ] **공유하고 싶은가?** - 친구에게 보여주고 싶은가?
- [ ] **바이럴 될 수 있는가?** - SNS에 올리고 싶은가?
- [ ] **템플릿 패턴을 사용하지 않았는가?** - "{주제} 사용 감정은?" 같은 패턴이 없는가?
- [ ] **일상적이고 친근한가?** - 누구나 경험할 수 있는 상황인가?
- [ ] **유쾌하고 긍정적인가?** - 부정적이거나 무거운 내용이 아닌가?

## 📌 질문 작성 단계별 가이드

### 1단계: 주제 선정
- 퀴즈의 핵심 주제를 명확히 파악

### 2단계: 일상적 상황 생각하기
- 해당 주제와 관련된 일상적이고 재미있는 상황들을 나열

### 3단계: 예상치 못한 상황 추가
- 일상적 상황에 예상치 못한 요소를 추가하여 재미있게 만들기

### 4단계: 공감대 형성
- "아, 나도 그런 적 있어!"라고 생각되는 상황인지 확인

### 5단계: 바이럴 가능성 확인
- 공유하고 싶은 내용인지 확인

### 6단계: 템플릿 패턴 제거
- "{주제} 사용 감정은?" 같은 패턴이 없는지 확인

## 🔥 바이럴 질문 작성 마법의 공식

**"일상 + 예상치 못한 상황 + 재미 = 바이럴 질문"**

예시:
- 일상: "요리 중"
- 예상치 못한 상황: "불이 꺼졌는데 재료는 다 넣었을 때"
- 재미: 웃음과 공감 유발
- = "요리 중 불이 꺼졌는데 재료는 다 넣었을 때" ✅
