# 퀴즈 질문 작성 규칙 (필수 준수)

## 🎯 목적
앞으로 개발할 모든 퀴즈가 개선된 질문 스타일을 유지하도록 보장하기 위한 명확한 작성 규칙

---

## 🚨 절대 금지 사항

### ❌ 템플릿 질문 패턴 (발견 시 즉시 삭제)

**"은?", "는?", "이?", "가?"로 끝나는 모든 질문은 템플릿 패턴입니다!**

```
❌ "연락은?"
❌ "톤은?"
❌ "기념일은?"
❌ "정산은?"
❌ "관리는?"
❌ "처리는?"
❌ "행동은?"
❌ "선택은?"
❌ "준비는?"
❌ "전략은?"
❌ "추가는?"
❌ "반응은?"
❌ "점검 포인트는?"
❌ "조합은?"
❌ "후기는?"
❌ "논쟁은?"
❌ "중시하는 것은?"
❌ "평가는?"
❌ "판단 기준은?"
❌ "모습은?"
❌ "역할은?"
❌ "한 마디는?"
❌ "남기는 것은?"
❌ "{주제} {행동}은?"
❌ "{주제} {행동}는?"
❌ "{주제} {행동}이?"
❌ "{주제} {행동}가?"
```

**⚠️ 이런 질문들은 즉시 삭제하고 재작성해야 합니다!**

---

## ✅ 필수 사용 패턴

### 올바른 질문 형식

**질문은 반드시 구체적인 시점과 상황을 묘사해야 합니다:**

```
✅ "{상황}에서 {행동}할 때"
✅ "{상황}에서 {행동}했을 때"
✅ "{상황}에서 {행동}일 때"
✅ "{상황} 중에 {행동}할 때"
✅ "{상황} 중에 {행동}했을 때"
✅ "{상황} 전에 {행동}할 때"
✅ "{상황} 후에 {행동}할 때"
✅ "{상황}에 {행동}했을 때"
✅ "{상황}에 {행동}일 때"
```

### 질문 길이 규칙
- **20자 이내 권장** (너무 길면 읽기 어렵고 재미가 떨어짐)
- 핵심 상황만 간단명료하게 표현
- 불필요한 수식어 제거

## 🎯 질문 작성 단계별 가이드

### 1단계: 주제 파악
- 퀴즈의 핵심 주제를 명확히 파악
- 예: "치킨 스타일", "편의점 루틴", "OTT 시청 습관"

### 2단계: 일상적 상황 나열
- 해당 주제와 관련된 일상적이고 재미있는 상황들을 나열
- 예: "치킨 주문", "치킨 배달", "치킨 먹기", "치킨 정산"

### 3단계: 구체적 시점 추가
- 각 상황에 구체적인 시점("~할 때", "~했을 때")을 추가
- 예: "치킨을 시키기 전에 메뉴를 정할 때"
- 예: "치킨이 도착했을 때"
- 예: "치킨을 같이 먹을 때"

### 4단계: 템플릿 패턴 검사
- 질문이 "은?", "는?", "이?", "가?"로 끝나지 않는지 확인
- 구체적인 시점을 포함하는지 확인
- 20자 이내로 간결한지 확인

### 5단계: 재미와 공감 확인
- 읽고 웃음이 나오는가?
- "아, 나도 그런 적 있어!"라고 생각되는가?
- 공유하고 싶은가?

## 📋 질문 작성 체크리스트

각 질문을 작성한 후 반드시 확인:

### 필수 검사 (통과하지 못하면 즉시 재작성)
- [ ] 질문이 "은?", "는?", "이?", "가?"로 끝나지 않는가?
- [ ] 질문이 구체적인 시점("~할 때", "~했을 때", "~일 때")을 포함하는가?
- [ ] 질문이 20자 이내로 짧고 간결한가?

### 품질 검사
- [ ] 재미있고 유쾌한가?
- [ ] 구체적인 상황을 묘사하는가?
- [ ] 공감대를 형성하는가?
- [ ] 공유하고 싶게 만드는가?
- [ ] 바이럴 될 수 있는가?

## 🔥 핵심 원칙

1. **템플릿 질문은 절대 사용하지 않는다**
   - "은?", "는?", "이?", "가?"로 끝나는 질문 금지

2. **구체적인 시점을 반드시 포함한다**
   - "~할 때", "~했을 때", "~일 때" 등

3. **짧고 간결하게 작성한다**
   - 20자 이내 권장

4. **구체적인 상황을 묘사한다**
   - 추상적이거나 모호하지 않게

5. **재미있고 공감되는 내용으로 작성한다**
   - 바이럴 될 수 있게

## 📌 개발 시 참고

**새로운 퀴즈를 개발할 때:**
1. 이 문서를 먼저 읽고 규칙을 숙지한다

3. 템플릿 패턴이 발견되면 즉시 재작성한다

**질문 검토 시:**
1. 모든 질문이 "은?", "는?", "이?", "가?"로 끝나지 않는지 확인
2. 모든 질문이 구체적인 시점을 포함하는지 확인
3. 모든 질문이 짧고 간결한지 확인
4. 모든 질문이 재미있고 공감되는지 확인
